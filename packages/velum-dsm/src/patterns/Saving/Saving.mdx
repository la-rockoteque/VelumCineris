# Saving

Saving states should preserve edit context. Keep the form readable, show the phase of the save, and disable only conflicting actions.

## Contract

- Leave field values visible while the save runs.
- Show save status close to the edited surface.
- Keep non-conflicting context readable so users do not lose orientation.

## Recommended Use Cases

- Record editors with autosave or manual publish
- Mutation flows with linked-field synchronization
- Integration actions that have a dry-run and live mode
