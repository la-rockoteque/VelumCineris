# Pagination

Pagination metadata and controls should live directly with the dataset they govern.

## Contract

- Keep the current range or page count visible at all times.
- Place navigation immediately below or beside the dataset.
- Do not separate page controls from the affected content region.

## Recommended Use Cases

- Server-backed row browsing
- Log and history pages
- Search results with stable result counts
