# Loading

Use the same shell, preserve the occupied space, and pause conflicting write actions while work is in flight.

## Contract

- Keep the user’s bearings by leaving the surrounding layout in place.
- Prefer local loading messaging inside the affected card or section.
- Disable only the actions that would conflict with the fetch or mutation.

- Reloading validation catalogs or translator context
- Background refresh after a filter or source switch
